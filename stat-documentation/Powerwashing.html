Powerwashing

Powerwashing a Chromebook will erase all locally saved data, including all user accounts currently on the device. Thanks to Chrome OS's robust cloud sync features, data loss is only a minor concern, making nuking the OS and starting over a more palatable option when troubleshooting.  

It is prudent to check that the user has no saved files in their Downloads folder (which is not synced to Google). If they do and wish to save them, simply sync them to Google Drive using Chrome OS's file manager. Use of the Chrome OS File Manager app is outside the scope of this documentation. Refer to the linked Google Support article if you require help with that piece of software.


Materials

  • External recovery disk (optional)
  • Fingers

Time: 1-3 minutes

To powerwash a Chromebook:


  1. Press the ESC+Refresh+Power keys in that sequence.
  2. When the powerwash screen appears, press CTRL+D:
  3. On the next screen, press space:
  4. Press enter
  5. Hit enter again.
  6. The device will reboot.


Go through the Chrome OS setup, enrolling the device if necessary.


If a normal powerwash does not fix a hardware issue, try a hard powerwash before moving on to the next step in diagnosis.


Hard powerwash

Chrome OS is hard to break, but it is possible. Thus, Chrome OS allows you to reload the OS from an external USB drive or SD card. This type of powerwash can also occasionally fix hardware problems that a normal one would not.


This is an uber-powerwash. The same caveats as before apply.


  1. Press ESC+Refresh+Power
  2. Insert the recovery flash drive. There are specific model images, so be sure to use the right kind. STAT's recovery media is well-labeled, so this should not be an issue.
  3. The recovery media will be verified and then the reload will occur. Check back every few minutes, as you have to remove the flash drive before the device will reboot.


After a hard powerwash, you will end up with a much older version of Chrome OS. If the issue you were experiencing was resolved, log in with the STAT account and update Chrome OS until the device is running the latest version.
